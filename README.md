# roblox-js

A Node.js library for interacting with the Roblox API. Easily build bots, scripts, and automation tools for Roblox games and services from JavaScript or TypeScript!

## Features

- **Seamless Roblox API Integration:** Get game information, manage user data, and automate tasks.
- **Robust Authentication:** Secure, session-based authentication for protected endpoints.
- **Easy-to-use Methods:** Well-organized functions for common Roblox operations.
- **Promise-based & Async/Await Support:** Designed for modern asynchronous workflows.

## Getting Started

### Installation

```bash
npm install roblox-js
```

### Usage

```js
import { CaptureService } from "@rbxjs/services"

CaptureService:CaptureScreenshot(onCaptureready) // onCaptureready is an example
```
